Oracle® JRockit Mission Control

Using the Triggers Tab

Release 4.1

E21585-01

Use the Triggers tab on the Mission Control Console to alert you when a specified event occurs or when a threshold is exceeded. This is a useful way track run-time issues without having to constantly monitor the application. This section contains the following topics:

  • Adding a New Trigger Rule Using the Add Wizard

  • Editing a Trigger Rule on the Triggers Tab

  • Editing a Trigger Rule Using the Edit Wizard

  • Deleting a Trigger Rule

  • Viewing Application Alerts

  • Using the Trigger Alerts Window
