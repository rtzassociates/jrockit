Graph and Type Representation are Now Combined

In earlier versions of JRockit Mission Control, the graph and tree representations for the Type Graph and the Instance Graph were separate tabs adjacent to each other in the Memory Leak Detector GUI. Since these are two different views of the same model, you can now switch between the two different views (graph and tree) on the same tab. Table: Type/Instance Graph-to-Tree Toggle shows which icon you click to toggle from one view to the other.

Type/Instance Graph-to-Tree Toggle

To... Click this Icon...

Toggle from the graph view to the tree view


Toggle from the tree view to the graph view



The Type Graph and Type Tree tabs are combined on the Type relations tab and the Instance Graph and Instance Tree are combined on the Instance relations tab.
