Memory Leak Detector Tool Bar

Table: Memory Leak Detector Tool Bar icons describes the different tools in the toolbar.

Memory Leak Detector Tool Bar icons

Tool Description Used On


The join classes button for trend analysis. Clicking this button joins all classes of the same name into one entry, regardless of the class loader that loaded the individual class.

Trend tab


The split classes button for trend analysis. Clicking this button splits all classes of the same name but with different class loaders into separate entries.

Trend tab


The Reset button. Resets the instance count delta and size delta for trend analysis.

Trend tab

The Refresh button. Refreshes the tree/table.

Trend tab

Allocation Tab


The Pause button. Pauses updating the tree/table, but continues analysis in the background.

Trend tab

Allocation tab

The join classes button on the Type relations tab. When this button is selected, new nodes are created only with regards to the class name and regardless of the class loader that loaded the individual class.

Type relations tab


The split classes button on the Type relations tab. When this button is selected, new nodes are created for each individual class, even if they have the same class name.

Type relations tab

The garbage collection button. Clicking this button performs garbage collection on the server.

All tabs


The table layout button. Clicking this button displays the Table Layout dialog box.

Trend tab

Allocation tab


The start button. Clicking this button starts trend analysis or allocation tracing.

Trend tab

Allocation tab


The stop button. Clicking this button stops the trend analysis or allocation tracing

Trend tab

Allocation tab


The zoom-in button. Clicking this button enlarges the image in the graph, showing more detail for fewer nodes; in effect, it zooms in to the image.

Type relations tab

Instance relations tab


The zoom-out button. Clicking this button reduces the image in the graph, showing more nodes; in effect, it zooms out of the image.

Type relations tab

Instance relations tab


The zoom to 100% button. Clicking this button restores the image in the graph to its original size.

Type relations tab

Instance relations tab


The zoom-to-fit. Clicking this resizes and repositions the image in the graph so that all nodes appear on the graph.

Type relations tab

Instance relations tab


The layout graph button. Clicking this button allows you to refresh the layout of the graph.

Type relations tab

Instance relations tab


The clear graph button. Clicking this button removes all nodes from the graph.

The tree-to-graph toggle. Toggles from the graph view to the tree view. This tool only appears when the graph view is displayed.

Type relations tab

Instance relations tab


The graph-to-tree toggle. Toggles from the tree view to the graph view. This tool only appears when the tree view is displayed

Type relations tab

Instance relations tab


The export graph button. Clicking this button allows you to export the displayed graph to an image file.

Type relations tab

Instance relations tab


The print graph button. Clicking this button prints the graph.
