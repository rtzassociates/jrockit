Using the Oracle JRockit Memory Leak Detector

The Memory Leak Detector is a plug-in to Oracle JRockit Mission Control that provides real-time profiling to detect memory leaks within Java applications running on the Oracle JRockit JVM. A memory leak occurs when an application holds on to memory that it is no longer using.
