Shortest Path to an Instance's Garbage Collection Root

You can now expand an instance to its garbage collection root from the Trend tab. In earlier versions of JRockit Mission Control, you could expand an instance to its garbage collection root only from the Instance Graph (now called the Instance relations tab). You can now do this directly from the Trend tab by using the following procedure:

  1. On the Trend tab, right-click the type for which you want to see the garbage collection root.

    A context menu appears.

  2. On the contex menu, select With Random Instance then Expand to Root.

    The Instance relations tab appears, showing the type, expanded to its root.

For releated information, see Finding the Garbage Collection Root.
