New, Resettable Instance and Growth Counters

The Trend Analysis table, on the Trend tab, now includes columns that show the instance count and memory growth. These columns, labeled Delta Count and Delta Size, are resettable; that is, by clicking the reset delta button:


you can reset all values in these columns to 0.

For related information, see Analyzing Live Set Trends and Customizing the Trend Tab.
