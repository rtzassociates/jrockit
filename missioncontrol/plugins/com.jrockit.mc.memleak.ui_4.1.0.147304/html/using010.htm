Starting the Investigation

To start investigating a leak, you need to identify the type where the leak might be occurring and add that type to the Type relations tab.

To start investigating a leak:

  1. Right-click a type from the Trend tab.

    The context menu opens.

  2. Select Add to Type Graph.

    The Type relations tab appears showing the selected type.

The Type relations tab offers a view of the relationships between all the types pointing to the type you are investigating. Note that for each type shown, you can expand its references to show the number of instances of other types that point to that type.

The color red (dark) means that the type has a high growth rate (which might indicate a memory leak).
