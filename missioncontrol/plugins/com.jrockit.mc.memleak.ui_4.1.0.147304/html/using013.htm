Displaying Instances

To display an instance on the Instance relations tab:

  1. On the Instance List, right-click the instance you want to display.

    A context menu appears.

  2. Select Add to Instance Graph.

To add class loader instances for specific types directly to the Instance relations tab:

  1. On the Type relations tab, right-click the type you want to analyze.

    A context menu appears.

  2. On the context menu, select With Class Loader then Add to Instance Graph.

For additional information on the Instance List, see "Using the Instance List".
