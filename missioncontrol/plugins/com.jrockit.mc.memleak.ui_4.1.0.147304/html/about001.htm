New Features and Updates in this Release

The Oracle JRockit Memory Leak Detector shipped with JRockit Mission Control 4.1 Client includes these new features and updates:

  • Extension Point for Instance Visualization

  • Shortest Path to an Instance's Garbage Collection Root

  • New Class Loader Context Menu

  • Graph and Type Representation are Now Combined

  • New, Resettable Instance and Growth Counters
