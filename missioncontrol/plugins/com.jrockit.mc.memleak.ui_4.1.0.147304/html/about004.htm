New Class Loader Context Menu

To enable you to start analyzing the class loader for specific types, you can now add class loader instances for specific types directly from the Type relations tab to the Instance relations tab and instance inspector. You can do this by using the following procedure:

  1. On the Type relations tab, right-click the type you want to analyze.

    A context menu appears.

  2. On the context menu, select With Class Loader then:

    • If you want to add the type to the Instance relations tab, select Add to Instance Graph.

      The Instance relations tab appears and the selected type appears on the Type Graph.

    • If you want to inspect the instance, select Inspect Instance.

      The Instance Inspector opens, showing the selected instance.

For releated information, see Displaying Instances.
