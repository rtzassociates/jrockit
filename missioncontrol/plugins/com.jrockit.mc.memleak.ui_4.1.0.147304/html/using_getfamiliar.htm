Getting Familiar with the Memory Leak Detector

When you start the Memory Leak Detector, it shows the Trend tab with the Trend Analysis activated.

At the top of the Memory Leak Detector tab is the toolbar, where you control the updating of the analysis, how to view the different graphs when drilling down to find the memory leaks, etc. For a detailed description of the toolbar, see Memory Leak Detector Tool Bar.

At the bottom of the window you can find tab controls for the different tabs in the Memory Leak Detector. For an introduction to these tabs, see "Memory Leak Detector Tabs".
