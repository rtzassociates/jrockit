Removing Instances from the Instance Relations Tab

If other instances have already been added to the graph, they will appear along with the instance you just added. If you want to see just the new instance (for example, because otherwise the graph will be to crowded or otherwise difficult to read, you need to remove the other instances. You can either remove selected nodes from the graph or you can clear the graph completely.

To remove selected nodes from the Instance relations tab:

  1. Hold down the left mouse button and drag the pointer over all the nodes you want to remove.

    A wire frame surrounds all selected nodes in the range until you release the mouse button, then the nodes turn grey.

  2. Right-click anywhere on the graph to open the context menu.

  3. Select Remove.

    The selected nodes will disappear from the graph.
