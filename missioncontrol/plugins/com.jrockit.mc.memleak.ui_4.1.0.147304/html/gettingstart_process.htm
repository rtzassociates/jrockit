Detection Process Used with the Memory Leak Detector

This topic gives you an overview of the process of detecting memory leaks using the Memory Leak Detector. For more detailed instructions, see "Using the Memory Leak Detector".

Memory leak detection consists of the following steps:

  1. Starting the Memory Leak Detector

  2. Trend Analysis

  3. Type Relations Study

  4. Instance Investigation
