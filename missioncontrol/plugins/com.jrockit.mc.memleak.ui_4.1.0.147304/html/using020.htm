Working with the Instance List

Once you have displayed a list of instances, you can select from a group of context menu commands to display it on the Instance relations tab or inspect it by using the Instance Inspector. You can also use the type of an instance and add that to the Type relations tab, list all instances for the type, list the largest arrays in the type, and trace the type allocations. This feature is useful once you've identified the instance and want to investigate it within the context of the type to which it belongs.

To work with the Instance List:

  1. Right-click the instance with which you want to work.

    The context menu appears.

  2. Select the command you want to execute, as described in Table: Working with the Instance List Context Menu.
