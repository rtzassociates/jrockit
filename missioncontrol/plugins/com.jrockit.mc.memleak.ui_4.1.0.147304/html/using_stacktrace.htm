Viewing Allocation Stack Traces

In the Allocation tab, you can check for where in the code allocations of a certain type are done. Be aware that enabling allocation stack traces may have a negative impact on the performance of the JRockit JVM.

To jump to application source code

Jump directly from the Allocation tab to application source code. For instructions, please refer to Jumping to Application Source.
