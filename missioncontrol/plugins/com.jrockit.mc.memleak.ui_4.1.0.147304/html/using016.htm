Viewing Complex Graphs

Occasionally, a Type relations tab or Instance relations tab will become so large that it renders the nodes too small for easy viewing. In this case, you can use the Bird's Eye View feature to focus on selected nodes.

To get an overview of the graph:

From the Window menu, select Show View then Bird's Eye View.

A small Overview window opens on the tab. To see selected nodes, slide the shaded area over those nodes. The Type relations tab will reposition its focus on those nodes.
