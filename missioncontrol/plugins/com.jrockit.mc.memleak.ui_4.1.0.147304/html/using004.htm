Adding and Removing Columns

When you open the Trend tab, only certain columns appear. You can add columns by doing the following:

  1. Right-click anywhere on the Trend table to open the context menu.

  2. Select Visible Columns to display a list of all available columns.

  3. Select the column you want to add.

  4. Repeat steps 1 through 3 for each column you want to add.
