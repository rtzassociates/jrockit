Changing the Sort Order and Sort Key

You can change the sort order from descending to ascending and back and, at the same time, select which column on which to sort the data. To do so, simply click the column by which you want to sort the data. This will toggle the sort order to ascending and use the selected column as the key; for example, if you select Type, it will sort all data by Type in either ascending or descending order, depending on the current order.
