Getting Started

Finding memory leaks in an application requires some detective work, since you might not always see the obvious leaks. The Memory Leak Detector can help you to locate memory leaks in your application, down to the faulty method.

This topic describes the memory leak detection procedure that you can easily adapt to your normal development environment, through the use of Oracle JRockit Mission Control.

This topic is divided into the following subtopics:

  • Spotting a Suspected Memory Leak

  • Detection Process Used with the Memory Leak Detector
